# movie-portal-data-fetcher

用于获取电影传送门的视频信息、原视频的信息并存入 Notion 中的命令行工具。

## Roadmap

- [x](获取B站上的视频列表)
- [x](根据B站视频的来源信息，获取相应的YouTube视频信息)
- [ ](根据B站视频的来源信息，获取相应的Vimeo视频信息)
- [x](把视频信息存入Notion数据库)
